# Kenny Portes Project 2

## Overview 
This project is a streamlined web application designed for browsing PlayStation 5 titles. Simply enter a
title into the search bar to explore a massive library of PlayStation 5 games. Browse through your 
results and click on any game to unlock deeper insights, gameplay info, and everything else you need to 
know before you play.

## Functionality
A search form on the web page to search video games made for Playstation 5. Where the user type in an 
input value the code fetches the api key and outputs it search for any related video game for PS5. Async 
is used to always return a promise.Await is also used in the code which pauses the execution of that 
function until a task finishes. As the code is fetch the data from API key it dispalys in a text what it
is doing (e.g. Searching...). It then will display an image of the game, as well as the 
rates and the relase date for that specific game.
